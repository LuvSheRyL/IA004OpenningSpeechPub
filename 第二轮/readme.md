- 互相校对文字内容
- 对提及到的一些黑话（术语）进行增补解释；（可以使用 site:www.yangzhiping)
- 对阳老师提及的一些过于抽象的理论或者黑话（术语）增补一个通用的案例；
- 用写匠检查错别字

以下是通过看文心书话的一点总结：

1. 减少“的”等助词的使用；
2. 恰当使用一些修辞（这个比较高阶）；
3. 从宏观检查文稿的结构。
